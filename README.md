# BlueLog
<p>A Blog implemented via Flask.</p>
<br>

## Features
- Post display (according to category)
- Theme changeable
- Comment to Post
- Comment reply
- Manage Post/Comment/Category/Link
- Rich Text in new Post
<br>

## Extensions
- flask-sqlalchemy
- flask-mail
- flask-moment
- bootstrap-flask
- python-dotenv
- flask-ckeditor
- flask-wtf
- faker
- flask-login
- flask-migrat
